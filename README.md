# PROTOS SOFTWARE

ESP32 Software for controlling visor

### CURRENTLY SUPPORTED DEVICES

- PX Matrix, tested on size 128x32 in 3-bit color mode
- MAX7219 Matrix, tested on 2xBulk2x2 and in Simple Visor mode
- DHT22
- INA219
- QC control for FANS
- OLED 128x32
- SD CARD
- WIFI CONTROL WITH PROTAPP
